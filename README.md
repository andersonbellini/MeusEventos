## Basic Setup

1. [Install Node.js](https://nodejs.org/)
1. [Install .NET Core 5](https://dotnet.microsoft.com/download/)
2. Fork the [MeusEventos](https://github.com/andersonbellini/MeusEventos/fork)
3. Clone the repo you just forked.
